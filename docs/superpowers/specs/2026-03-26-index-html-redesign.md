# index.html リデザイン仕様

## 概要

既存の `index.html`（紫グラデーション背景＋白カード）を、グラスモーフィズム×オーロラカラーのデザインに刷新する。

## デザイン方針

- **スタイル:** グラスモーフィズム（半透明ガラス風UI）
- **背景:** オーロラグラデーション（`#a18cd1 → #fbc2eb → #a6c1ee`、135deg）
- **カード:** 白みを強めた半透明（`rgba(255,255,255,0.72)`）＋ `backdrop-filter: blur(24px)`

## カラーパレット

| 用途 | 値 |
|------|-----|
| 背景グラデーション | `#a18cd1 → #fbc2eb → #a6c1ee` |
| カード背景 | `rgba(255,255,255,0.72)` |
| タイトル・本文テキスト | `#2d1b4e`（ダークパープル） |
| サブテキスト・ラベル | `#9b7fc0` |
| 追加ボタン | `linear-gradient(135deg, #a18cd1, #c084fc)` |
| 削除ボタン | ピンク系（`rgba(251,194,235,0.4)`） |
| チェックボックス（未完了） | ボーダー `#c084fc` |
| チェックボックス（完了） | グラデーション塗り `#a18cd1 → #c084fc` |

## 変更対象

`index.html` のCSSのみ変更。JavaScriptのロジックは一切変更しない。

## 変更内容

- `body` の背景をオーロラグラデーションに変更
- `.container` のスタイルを半透明ガラスカード（`backdrop-filter`使用）に変更
- テキスト色をダークパープル系に統一（白文字→濃色文字）
- ボタン・入力欄・タスク項目のスタイルをグラス風に変更
- 完了タスクの打ち消し線色を `#9b7fc0` に変更

## 変更しないもの

- HTMLの構造（タグ・クラス名・ID）
- JavaScriptのロジック（追加・削除・完了・localStorage）
- ファイル構成
